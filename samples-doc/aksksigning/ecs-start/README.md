# Start ECS instance using AK/SK signing

## Environment variables

| Env variable      | Value
| ----------------- | -----------------
| OTC_SDK_PROJECT_ID| Project ID
| ECS_INSTANCE_ID   | ECS Instance ID
| OTC_SDK_AK        | Access Key
| OTC_SDK_SK        | Secret Key

## Install

```bash
npm install
```

## Start

```bash
node startECS.js
